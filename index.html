Once upon a time there was a dog.

It was a very happy dog.

It was happy, because it was allowed to be a dog.

It was happy, because it didn't try to be anyone else.

It was happy, because it was ok being itself.

It didn't try to be a cat.

It didn't try to be a duck.

It was just all too happy being a dog.

Jumping through the fields...

Hunting squirrels in the forest...

Rolling through the mud...

Jumping into the river...

Chasing that wooden stick...

All the dog things that dogs do...

And it was happy, oh, so happy, because it didn't try to be anything else than what it was.

In every single fibre, it felt exactly as itself.

With every single fibre, it felt dog. It lived dog. It breathed dog.

Be - who - you - are.
